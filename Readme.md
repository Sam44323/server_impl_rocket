# **Scripts**

### Cloning the repo

```bash
git clone https://github.com/Sam44323/server_impl_rocket.git
```

---

_(Optional) Install cargo watch if you dont have it_

```bash
cargo install cargo-watch
```

---

### Installing the dependencies

```bash
cargo install
```

---

### Running the server

```bash
cargo watch -x run
```
